## 2.13.1 添加好友（含申请审批）

---

#### TC-2.13.1-01

**用例名称**：通过手机号发送好友申请（含留言）

**前置条件**：
- 用户已登录
- 设备联网正常，服务器可访问
- 目标手机号已注册

**操作步骤**：
1. 进入好友管理页面（通过主页人形图标或排行榜好友管理按钮）
2. 在「输入手机号」输入框中输入目标用户的 11 位手机号
3. 在「留言（选填）」输入框中输入「我是小明」
4. 点击「申请」按钮

**预期结果**：
1. 好友管理页面正常显示
2. 输入 11 位手机号后「申请」按钮变为可点击状态
3. 申请发送成功后显示提示文字「好友申请已发送！」
4. 对方的好友请求列表中出现该申请（含留言内容）
5. 双方尚未成为好友，需对方同意后才生效
