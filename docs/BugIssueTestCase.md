## v1.2.5 — 2.2.1 任务设定三个遗留缺口修复

**发布日期**：2026-03-03
**关联提交**：`70967b2`
**修复范围**：任务描述字段 / 任务复制 UI / 自定义重复规则 UI

---

### Gap 1：任务描述字段

#### TC-1.1 新建任务时填写描述

**前置条件**：打开每日任务列表页

**操作步骤**：
1. 点击右下角 FAB（`+`）→「新建任务」进入编辑页
2. 在「任务名称」填写"数学作业"
3. 观察「任务名称」输入框下方是否出现「任务说明（选填）」多行文本框
4. 点击该文本框，输入"第12页第1-5题，注意写步骤"
5. 点击右上角「保存」

**预期结果**：
- 保存成功，返回任务列表，无报错
- 任务说明内容持久化（下一步 TC-1.2 验证）

- [ ] 通过

---

#### TC-1.2 编辑已有任务时描述回显

**前置条件**：TC-1.1 已完成，"数学作业"已存在

**操作步骤**：
1. 点击「数学作业」任务卡片，进入编辑页
2. 观察「任务说明（选填）」文本框内容

**预期结果**：
- 文本框显示"第12页第1-5题，注意写步骤"（与保存时一致，不丢失）

- [ ] 通过

---

#### TC-1.3 描述为空时正常保存

**操作步骤**：
1. 新建任务，填写标题，不填写任务说明
2. 点击「保存」

**预期结果**：
- 保存成功，无错误提示
- 重新进入编辑页，说明文本框为空（不显示乱码或 null）

- [ ] 通过

---

#### TC-1.4 描述支持多行显示

**操作步骤**：
1. 新建任务，在「任务说明」输入 3 行以上内容（可用换行）
2. 保存后重新进入编辑页

**预期结果**：
- 输入框显示最少 2 行高度，超出内容可滚动（最多显示 4 行）
- 多行内容原样回显，换行保留

- [ ] 通过

### Gap 2：任务复制 UI

#### TC-2.1 FAB 展开后显示「复制」按钮

**前置条件**：当日任务列表有至少 1 条 PENDING 任务

**操作步骤**：
1. 点击右下角主 FAB（`+`），不要立即点击
2. 观察展开后的次级按钮列表

**预期结果**：
- 展开后显示至少三个次级按钮：「模板」、「添加里程碑」、「复制」
- 各按钮标签清晰可见

- [ ] 通过

---

#### TC-2.2 点击「复制」弹出日期选择器

**操作步骤**：
1. FAB 展开后点击「复制」小按钮

**预期结果**：
- FAB 收起
- 弹出系统日历样式的 DatePickerDialog，显示当前月份
- 对话框底部有「取消」和「复制」两个按钮

- [ ] 通过

---

#### TC-2.3 选择目标日期完成复制

**前置条件**：今天有 2 条 PENDING 任务、1 条已完成任务

**操作步骤**：
1. 点击 FAB →「复制」，弹出日期选择器
2. 选择明天的日期
3. 点击「复制」确认

**预期结果**：
- 对话框关闭
- 底部 Snackbar 显示：「已复制 **2** 项任务到 \<明天日期\>」（只复制 PENDING，已完成不复制）
- 切换到明天日期（点击 TopBar 右箭头），确认两条任务已出现

- [ ] 通过

---

#### TC-2.4 取消日期选择不执行复制

**操作步骤**：
1. 点击 FAB →「复制」，弹出日期选择器
2. 选择任意日期后点击「取消」

**预期结果**：
- 对话框关闭，无 Snackbar，无任何任务被复制

- [ ] 通过

---

#### TC-2.5 当日无任务时复制结果

**前置条件**：今天没有任何任务

**操作步骤**：
1. 点击 FAB →「复制」，选择目标日期，点击「复制」

**预期结果**：
- Snackbar 显示：「已复制 **0** 项任务到 \<目标日期\>」
- 目标日期无新增任务

- [ ] 通过

### Gap 3：自定义重复规则 UI

#### TC-3.1 新建任务时选择「自定义」重复规则

**操作步骤**：
1. 进入新建/编辑任务页
2. 在「重复规则」区域观察选项列表

**预期结果**：
- 显示 4 个 RadioButton：不重复、每天、周一至周五、**自定义**
- 默认选中「不重复」

- [ ] 通过

---

#### TC-3.2 选择「自定义」后显示星期 Checkbox

**操作步骤**：
1. 点击「自定义」RadioButton

**预期结果**：
- 「自定义」被选中（radio filled）
- 下方立即出现一行 7 个 Checkbox，标签分别为：周一、周二、周三、周四、周五、周六、周日
- 全部默认未勾选

- [ ] 通过

---

#### TC-3.3 勾选指定星期并保存

**操作步骤**：
1. 选择「自定义」
2. 勾选「周一」「周三」「周五」
3. 填写任务标题后点击「保存」

**预期结果**：
- 保存成功，无报错
- 任务卡片上显示「🔄」重复标记

- [ ] 通过

---

#### TC-3.4 编辑时自定义规则回显

**前置条件**：TC-3.3 已保存"周一、周三、周五"自定义规则任务

**操作步骤**：
1. 点击该任务卡片进入编辑页
2. 查看「重复规则」区域

**预期结果**：
- 「自定义」RadioButton 被选中
- 「周一」「周三」「周五」的 Checkbox 显示已勾选（✓）
- 「周二」「周四」「周六」「周日」未勾选

- [ ] 通过

---

#### TC-3.5 切换回其他规则后 Checkbox 收起

**操作步骤**：
1. 在编辑页已选中「自定义」、Checkbox 可见
2. 点击「每天」RadioButton

**预期结果**：
- 「每天」被选中
- 星期 Checkbox 行立即消失（不再显示）
- 再次点击「自定义」，Checkbox 重新出现

- [ ] 通过

---

#### TC-3.6 自定义规则不选任何星期时的边界行为

**操作步骤**：
1. 选择「自定义」但不勾选任何星期
2. 保存任务

**预期结果**：
- 保存成功（不强制校验）
- 该任务只在创建当日出现，之后无重复（等价于"不重复"，合理边界值）

- [ ] 通过
