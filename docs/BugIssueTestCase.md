### Gap 1：任务描述字段

#### TC-1.1 新建任务时填写描述

**前置条件**：打开每日任务列表页

**操作步骤**：
1. 点击右下角 FAB（`+`）→「新建任务」进入编辑页
2. 在「任务名称」填写"数学作业"
3. 观察「任务名称」输入框下方是否出现「任务说明（选填）」多行文本框
4. 点击该文本框，输入"第12页第1-5题，注意写步骤"
5. 点击右上角「保存」

**预期结果**：
- 保存成功，返回任务列表，无报错
- 任务说明内容持久化（下一步 TC-1.2 验证）

- [x] 通过
