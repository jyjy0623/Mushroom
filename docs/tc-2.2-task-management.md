#### TC-2.2.1-01

**用例名称**：正向流程——成功创建一个基础任务（必填字段）

**前置条件**：
- App 已启动，当前处于每日任务列表页
- 当天没有任何任务

**操作步骤**：
1. 切换到明天日期（点击 TopBar 右箭头），点击右下角浮动按钮（FAB）展开菜单
2. 点击 FAB 主按钮（"+"号）进入新建任务页
3. 确认「任务日期」字段默认显示明天的日期
4. 在「任务名称 *」输入框中输入「完成数学练习题」
5. 在「学科」下拉框中选择「数学」
6. 在「预计时长（分钟）」输入框中输入「30」
7. 点击右上角「保存」按钮

**预期结果**：
1. 保存按钮点击后，页面自动返回到每日任务列表
2. 任务列表中出现新建的任务卡片，显示任务名「完成数学练习题」
3. 任务卡片副标题行显示「数学」学科标签（Chip）
4. 任务卡片底部显示奖励预览文字（如「🍄 小蘑菇 × 1」）
5. 切换到明天日期，可见该任务出现在明天的任务列表中

- [x] 通过
> 不通过时记录：
> - 实际结果：新建任务页面，每次新建任务都是当天的任务，希望修改。1）新建任务默认的任务的日期是任务页面的时间，比如今天是3月4日，如果在3月5日界面进入新建任务，那这个任务就创建到3月5日。如果今天是3月4日，对于已经过去的时间的界面，比如在3月3日，则禁止新建任务，把FAB主按钮的（"+"号）去掉。2）新建任务界面，请增加可以设定日期的配置项，默认值参考上面的规则。
> - 设备 / Android 版本 / App 版本：
> - 截图或日志：（可选）
> - Issue：#33
