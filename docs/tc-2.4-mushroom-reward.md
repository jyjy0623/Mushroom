## 2.4.2 蘑菇获取规则

---

#### TC-2.4.2-01

**用例名称**：完成单项普通任务获得小蘑菇×1

**前置条件**：
- 当日任务列表有 1 条未完成的普通任务（无模板类型、无自定义奖励配置）
- 当前小蘑菇余额为 N

**操作步骤**：
1. 进入每日任务列表
2. 点击该任务的「完成打卡」按钮
3. 确认打卡成功
4. 进入「蘑菇账本」查看余额变化

**预期结果**：
1. 打卡成功，任务状态变为「已完成」
2. 蘑菇账本余额：小蘑菇变为 N+1
3. 流水记录中新增一条「+1 小蘑菇」记录，备注为「完成任务「XXX」」
4. 若为当日最后一项任务完成，额外触发全勤奖（中蘑菇×1）

- [x] 通过
> 不通过时记录：
> - 实际结果：若为当日最后一项任务完成，未额外触发全勤奖（中蘑菇×1），同时新增需求，把全勤奖，奖励一个中蘑菇的提示，显示在置顶的”今日进度”上，以提示用户，如果达成，也保留显示。
