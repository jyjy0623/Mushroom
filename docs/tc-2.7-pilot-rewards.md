## 2.7.2 看电视奖励

---

#### TC-2.7.2-01

**用例名称**：看电视奖品列表显示与基础信息

**前置条件**：
- 家长已在奖品库创建"看电视"时长型奖品，配置：30 分钟/次需小蘑菇×3，每周上限 90 分钟
- 当前周尚未兑换任何看电视时长
- 学生蘑菇余额：小蘑菇 ≥ 3

**操作步骤**：
1. 打开 App，进入底部导航「奖品」页
2. 在奖品卡片列表中找到「看电视」奖品
3. 观察卡片上显示的信息
4. 点击「看电视」奖品卡片，进入详情页
5. 观察详情页「本期额度」区域

**预期结果**：
1. 奖品列表中「看电视」卡片图标显示为 ⏱（时长型标志）
2. 卡片上显示本周已使用时长与最大时长（如「0/90 分钟」）
3. 进入详情页后，顶部标题显示「看电视」
4. 「本期额度」卡片显示「0 / 90 分钟」以及进度条（进度为 0%）
5. 显示「剩余 90 分钟」字样
6. 兑换区域可见「消耗蘑菇」卡片，包含蘑菇等级选择按钮和数量调节按钮

- [ ] 通过
