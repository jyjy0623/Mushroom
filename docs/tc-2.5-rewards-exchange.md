---

#### TC-2.5.1-01

**用例名称**：家长创建实物型奖品（拼图类型）

**前置条件**：
- 家长已通过 PIN 验证
- 当前奖品列表为空

**操作步骤**：
1. 进入「奖品」列表页
2. 点击右下角 FAB（+）按钮，进入「创建奖品」页面
3. 填写奖品名称「高达模型」
4. 选择奖品类型「实物奖品（拼图解锁）」（RadioButton）
5. 在「兑换所需蘑菇」区域选择等级「金蘑菇（✨）」，数量填写「1」
6. 在「拼图设置」区域填写拼图总块数「20」
7. 点击右上角「保存」

**预期结果**：
1. 保存成功，页面返回奖品列表
2. 奖品列表中出现「高达模型」卡片，类型图标为 🎁（实物型）
3. 卡片显示「0/20 块」及空进度条（0%）
4. 不崩溃

> - 实际结果：高达模型创建阶段，建议提供用户可以输入的奖品的图片的UI。
